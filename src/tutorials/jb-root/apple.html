What are the cons?

Some banking apps can detect jailbreak and refuse to run. Besides that, downsides of jailbreaking is pretty rare. If you run into issues you can easily(-ish) undo everyting anyways.

Backup everyting before you start

Go to the settings and do a manual backup. Make sure to have a up to date backup before you start because this process might include resetting your phone (Don't worry. iCloud backups absolutely everything on your phone. You won't lose anything)

Is jailbreaking software trustworthy?

The cool hacker mode scrolling text might scare you but don't worry. Jailbreaking software mentioned in ios.cfw.guide is free and open source. Meaning it's completely community-developed anyone can look at what's inside it.

What is Open Source and FOSS (free and open source)?

Avoid using anything that is not mentioned in the cfw.guide. If cfw.guide says it is not possible yet, then it's not possible yet and you need to wait.
